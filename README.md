# Intergalactic Cargo Portal

A full stack application built as part of the Intergalactic Cargo Portal assessment.

## Tech Stack

### Frontend

- React
- React Router
- Axios
- React Toastify

### Backend

- Node.js
- Express.js
- JWT Authentication
- Multer

### Database

- MongoDB

### LOCAL SETUP INSTRUCTIONS

### Backend

Navigate to the backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add the required values:

```env
PORT=7777
MONGO_URI=mongodb+srv://rohithreddykuruganti_db_user:w8SXO3K5QavoOflt@cargocluster.wewxatp.mongodb.net/intergalactic-cargo
JWT_SECRET=mykey0409
```

Start the backend server:

```bash
npm run dev
```

Backend runs on:

```text
http://localhost:7777
```

### Frontend

Navigate to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the React application:

```bash
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
